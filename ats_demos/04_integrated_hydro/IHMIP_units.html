Workbook for the Integrated Hydrology Model Intercomparison #1, after Sulis et al 2010 and Maxwell et al 2013.

[1]:

# units translations
rho = 997.0
M = 18.0135e-3
visc = 8.9e-4
g = 9.80665
min_s = 60.

[4]:

# original paper units: saturated limited run
n_liq = rho / M; print "molar density = ", n_liq

# convert conductivity to permeability
k_sat = 6.94e-4  # m / min
rainfall_rate = 3.3e-4 # m / min
alpha = 1.0 # 1 / cm
Ss = 5.e-4 # 1 / m
n_mann = 3.31e-3 # m^-1/3 min

# ats units
K = k_sat / min_s * visc / (g * rho); print "K = ", K
Pr = rainfall_rate / min_s; print "Pr = ", Pr
alp = alpha * 100 / (rho * g); print "alpha = ", alp
Sstore = Ss / (rho * g); print "compressibility = ", Sstore
n_m = n_mann * 60; print "manning coeff = ", n_m

molar density =  55347.3783551
K =  1.05288852576e-12
Pr =  5.5e-06
alpha =  0.0102278456668
compressibility =  5.11392283339e-08
manning coeff =  0.1986

Note that with this alpha, the entire top meter is basically at residual saturation. At that residual value, porosity, and depth, the volume of space available to fill is greater than the rainfall.

Instead, a good match with papers is seen if alpha is assumed to have been intended to be in [1/m], not [1/cm].

[5]:

alp = alpha / (rho * g); print "alpha = ", alp

alpha =  0.000102278456668
